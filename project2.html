📘 作品二：Teachable Machine + TensorFlow 即時影像辨識

📘 內容說明

  • 分類標籤： 模型預測三個類別：mouse、phone、bottle。
  • 即時預測結果： 畫面左側顯示每個類別的機率：
    • phone: 0.5770754（即 57.7% 的機率是手機）
    • bottle: 0.0151199
    • mouse: 0.0066953
  • 預測判定： 模型成功辨識出使用者手中物品為手機，並以大字藍色「phone」標註於畫面上方。

🧠 程式設計說明

這段程式碼使用 TensorFlow 讀取 Teachable Machine 匯出的 .h5 模型，搭配 OpenCV 實作即時辨識。程式邏輯包含：

  • load_model() 載入訓練完成的模型（keras_model.h5）。
  • 使用 VideoCapture() 開啟攝影機持續擷取畫面。
  • 將影像轉為 224x224 並正規化處理。
  • 用 model.predict() 預測畫面中物件類別的機率。
  • 透過 cv2.putText() 將預測類別與機率顯示於畫面。

判斷邏輯： 若某類別的機率 > 0.5，即判定為該物件，並顯示於畫面（如：phone 為藍色）。

  • 即時辨識應用潛力： 即使是簡化平台訓練的小模型，辨識結果依然流暢，展現互動應用潛力。
